L'objectif de ce projet est d'implémenter le jeu du 2048 découper en 4x4 ou l'on doit combiner les cases pour a la fin obtenir une case à 2048.

pour compiler: javac better.java
pour executer: java better
ou avec le .jar: java -jar Jar2048.jar


vous pouvez déplacer les cases avec les flèches directionnel et en appuyant sur la barre espace vous pouvez relancer une nouvelle partie, votre score sera sauvegardé seulement à la fin d'une partie.
